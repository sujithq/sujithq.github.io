Azure Developer CLI (azd) now has a public preview for observability developer experience, aimed at adding an evaluation loop to agents created with Microsoft Foundry. The preview supports hosted-agent lifecycle workflows in azd, including creating, provisioning, deploying, and initializing evaluation assets.
