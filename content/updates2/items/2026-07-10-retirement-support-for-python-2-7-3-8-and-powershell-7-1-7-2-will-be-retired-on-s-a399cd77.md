Azure Automation will stop supporting Python 2.7, Python 3.8, and PowerShell 7.1/7.2 starting October 1, 2026. Runbooks may still run, but these runtimes will no longer receive security updates or bug fixes.

- **Source:** [Azure Updates](https://azure.microsoft.com/updates?id=567556)
