AKS release 2026-05-29 rolls out Kubernetes 1.36 preview and patch updates for 1.35.5/1.34.8/1.33.12, with multiple security and component updates. It also adds Windows Server 2025 and Azure Container Linux GA, expands AKS Automatic and LocalDNS behavior changes, and includes several retirement/deprecation notices for older Windows and Flatcar node images.
