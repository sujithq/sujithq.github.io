GitHub says it will deprecate GPT-5.2 and GPT-5.2-Codex across GitHub Copilot experiences, including Chat, inline edits, ask and agent modes, and code completions. The notice says GPT-5.2-Codex is excepted in Copilot, but no replacement or timeline details are included in the provided text.
