Azure Site Recovery now supports replication and disaster recovery for Linux Azure VMs with NVMe disk controllers in Azure-to-Azure scenarios. The feature is in public preview and applies to NVMe-enabled Generation 2 VM families such as Da/Ea/Fa v6 and Ebsv5/Ebdsv5.
