April 2026 Microsoft Foundry updates include Foundry Local GA for local AI development, GPT-5.5 support with default Tier 5 and Tier 6 quota, and new tracing, monitoring, and evaluation features for agents. The release also adds CodeAct with Hyperlight for sandboxed code execution and SDK updates for Python, JavaScript/TypeScript, .NET, and Java.
